# Image assets

Drop all site images in this folder (`assets/img/`).

Reference them with a relative path from the site root, e.g.:

    cover:  "assets/img/ssd-cover.jpg"
    images: ["assets/img/ssd-01.jpg", "assets/img/ssd-02.jpg"]
    photo:  "assets/img/portrait.jpg"        // Info page
    { image: "assets/img/badge.png", caption: "…" }  // sidebar entry

Tips:
- Keep filenames lowercase with hyphens, no spaces (e.g. `lam-family-hero.jpg`).
- Optional: group by project with subfolders like `assets/img/ssd/01.jpg`.
- Web-friendly formats: .jpg / .png / .webp for images, .mp4 / .webm for video.
